=========
PY-LIBELF
=========

Python bindings for ``libelf`` which is part of elfutils, https://sourceware.org/elfutils/

.. image:: https://github.com/sonals/py-libelf/actions/workflows/Pipeline.yml/badge.svg

-------------------------------------------------------------------------------

Install
*******

.. code-block:: bash

        cd build
        cmake ../
        make


Test
****

.. code-block:: bash

        cd build
        cmake ../
        make install
        make test
